### 🕒 指导时间与效率

- _指导时间_：老师通常会在工作日的 **9:00 - 18:00** 集中回复问题
- _请理解_：老师并非24小时在线，若未及时回复，请耐心等待
- _效率秘诀_：提问要一次性说清楚！尽量将所有问题整理好，一次性发送，避免零散消息刷屏

### 📋 文件与通知管理

- 所有重要的通知、模板、指导资料等，都会上传到群文件
- 请大家养成定期查看群文件的习惯，重要文件请 $\color{blue}{\textbf{自行下载备份}}$
- 收到老师 @全体成员 通知时，请 $\color{blue}{\textbf{务必回复“收到”}}$，确保信息传达到位

### ⏳ 严格遵守截止日期

- 毕设过程中的所有阶段性任务（开题报告、中期检查、论文初稿等）都有严格的截止日期
- 所有提交都将以系统时间为准，逾期未交或迟交将影响你的毕设进程和成绩评定

### 📚 文档规范要求：格式决定印象分 ✨

- _外文翻译_：允许使用翻译工具进行辅助性翻译，但 **严禁将机翻结果未经人工校对修改直接提交**
- _译文要求_：专业术语准确无误，语句通顺，符合中文表达习惯
- _统一工具_：统一使用 $\color{red}{\textbf{Microsoft Word}}$ 进行毕业论文的撰写与排版
- _统一模板_：所有文档（开题报告、任务书、中期检查、论文初稿等）必须严格使用群文件或学校官网提供的最新官方模板
- _格式要求_：务必仔细阅读并遵守模板中关于字体、字号、行距、页眉页脚、目录生成的所有格式要求
- _图表清晰_：文档中的所有图表必须清晰可辨，有完整的图/表序号和标题，并在正文中进行引用和解释
- _参考文献_：参考文献格式（如引用顺序、标点符号、作者名、年份等）必须严格遵守学校规定的统一国标格式
- _图纸模型_：图纸必须符合国标，包括线型、标注、公差、粗糙度等。不能只交三维图，还要有二维工程图（装配图+零件图）
- _资料备份_：⚠️**注意数据安全**，U盘、电脑、网盘多处备份，“电脑坏了”、“文件丢了”不是理由

### 📌 沟通与纪律要求

- _沟通频率_：每两周汇报一次进度
- _沟通方式_：线上发简报，或者线下面对面
- _汇报内容_：这两周完成了什么？遇到了什么问题？接下来计划做什么？
- _请假制度_：如果遇到考研复试、考公、生病等情况，必须提前请假说明
